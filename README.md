A premium, high-performance personal portfolio built with a focus on modern aesthetics, smooth micro-interactions, and professional storytelling. Featuring a stunning glassmorphic interface and fully responsive layouts.

---

## ✨ Key Features

- **🔍 Dynamic Showcase**: Interactive project display with animated borders and hover effects.
- **⚡ Performance First**: Built with Vite and React for near-instant load times and high efficiency.
- **🎨 Glassmorphic Design**: Modern UI utilizing transparency, blur, and subtle gradients for a premium feel.
- **🎭 Smooth Animations**: Powered by Framer Motion for natural transitions and entrance effects.
- **📱 Fully Responsive**: Optimized for all screen sizes, from mobile devices to ultra-wide monitors.
- **🛠️ Professional Components**: Built with Radix UI and Shadcn primitives for accessibility and consistency.

- **Frontend Framework**: [React 18](https://reactjs.org/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/souvikx18/Portfolio_website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd My-Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
